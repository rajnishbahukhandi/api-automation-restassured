# API Automation Framework

A REST API Automation Framework built using **Java**, **Rest Assured**, **TestNG**, and **Maven** for testing RESTful APIs.

## 🚀 Tech Stack

- Java 21
- Rest Assured
- TestNG
- Maven
- JSON Path
- Jackson (POJO Serialization & Deserialization)
- Git & GitHub

## 📁 Project Structure

```
src
 ├── test
 │    ├── java
 │    ├── resources
 ├── main
 │    ├── java
 │    ├── resources
pom.xml
testng.xml
README.md
```

## 📌 Features

- API CRUD Testing
- Request & Response Validation
- Status Code Validation
- Header Validation
- JSON Schema Validation
- Serialization & Deserialization
- Authentication
- Data-Driven Testing
- Reusable Framework Design
- Logging & Reporting

## 🛠 Prerequisites

- Java 21+
- Maven
- IntelliJ IDEA
- Git

## ▶️ Run Tests

Clone the repository:

```bash
git clone https://github.com/<your-username>/api-automation-framework.git
```

Navigate to the project:

```bash
cd api-automation-framework
```

Run all tests:

```bash
mvn test
```
